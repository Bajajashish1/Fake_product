## 🔑 **API Setup**

### **Step 1: Get Your Gemini API Key**
1. Go to [Google AI Studio](https://aistudio.google.com/)
2. Sign in with your Google account
3. Create a new API key
4. Copy the key (starts with `AIza...`)

### **Step 2: Configure in Dashboard**
1. Open: http://localhost:8505
2. In the **sidebar**, find "🤖 Gemini API Configuration"
3. Paste your API key in the "Gemini API Key" field
4. You'll see "✅ API Key configured!" when successful

## 🎯 **How to Use**

### **Option 1: Gemini API Only (Recommended for Shoes)**
1. Select **"🤖 Gemini API Only"** in Analysis Method
2. Upload your shoe/product image
3. Get comprehensive AI analysis including:
   - **Authenticity Assessment** (Authentic/Fake/Uncertain)
   - **Confidence Score** (0-100%)
   - **Quality Analysis** (Logo, Material, Construction)
   - **Detected Issues** (List of problems found)
   - **Detailed Recommendations**

### **Option 2: Compare Both AI Systems**
1. Select **"🔄 Both (Compare Results)"** 
2. Upload your image
3. See side-by-side comparison:
   - **YOLO Detection** (left side)
   - **Gemini Analysis** (right side)
   - **Agreement Analysis** (do they agree?)

### **Option 3: YOLO Only**
1. Select **"🎯 YOLO Model Only"**
2. Traditional object detection (your trained model)

## 📸 **Best Results Tips**

### **For Shoe Analysis:**
1. **Show the whole shoe** including sole and brand labels
2. **Good lighting** - natural light works best
3. **Clear focus** - ensure image is sharp
4. **Multiple angles** - take several photos if needed
5. **Brand details** - include logos, tags, labels clearly

### **Image Requirements:**
- **Formats:** JPG, JPEG, PNG
- **Size:** Will be automatically resized for API
- **Quality:** Higher quality = better analysis

## 🔧 **Troubleshooting**

### **"API Key not provided"**
- Make sure you entered the API key in the sidebar
- Check that it starts with "AIza" (Google AI keys)
- Ensure no extra spaces before/after the key

### **"API request failed"**
- Check your internet connection
- Verify your API key is valid
- Make sure you have API usage quota remaining

### **"No analysis generated"**
- Try a different image format
- Ensure image is not corrupted
- Check image size (very large images may fail)
